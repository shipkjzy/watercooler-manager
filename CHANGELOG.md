# Changelog

## 0.1.0

### Added
- 管理员权限自动申请
- 风扇自动曲线多坐标点支持
- 水泵自动曲线多坐标点支持
- 温控 RGB 联动
- 深色 / 浅色 / 跟随系统主题
- 自动模式防抖参数
  - 温度回差
  - 防抖采样次数
  - 风扇最短启停间隔
  - 水泵最短启停间隔
- 首次启动默认配置优化
- 配置文件自动保存与加载

### Changed
- 风扇显示改为百分比
- 风扇拖动精度改为 1%
- 水泵最大电压限制为 11V
- 自动曲线默认改为 4 点
- 默认主题改为跟随系统
- 自定义更新速度输入框仅在选择“自定义”时显示
- 风扇与水泵最短启停间隔改为分别设置

### Fixed
- RGB 温控颜色保存/读取异常
- 初始化阶段误触发配置覆盖问题
- 托盘与窗口状态问题
- 自动模式边界温度抖动导致的频繁启停问题
